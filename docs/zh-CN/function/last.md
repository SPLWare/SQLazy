#### function: last
Syntax: last({<data>})
Return value: numeric. Returns the last among multiple data/expressions.
Parameter: **<data>** original value/original expression. Usually multiple parameters used together, i.e., number of parameters >= 1. Required parameter; type is simple data type; parameter name omitted.
> Example: Take the last of "Class 2", "Class 3", "Class 3".
NLC snippet: last("Class 2","Class 3","Class 3") // returns "Class 3"
