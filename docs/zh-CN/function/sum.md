#### function: sum
Syntax: sum({<data>})
Return value: numeric. Returns the sum of multiple numeric values.
Parameter: **<data>** original value/original expression. Usually multiple parameters used together, i.e., number of parameters >= 1. Required parameter; type is numeric; parameter name omitted.
> Example: Sum 250, 3100, and the number of orders this year.
NLC snippet: sum(250,3100,order_count_this_year)
