#### function: quarter_end
Syntax: <object_parameter> quarter_end
Return: last day of the quarter containing the date <object_parameter>.
Parameter **<object_parameter>**: Required parameter; type is date or datetime; parameter name omitted.
> Example: Last day of the quarter containing 1980-02-27.
NLC snippet: 1980-02-27 quarter_end // result is date 1980-03-31
