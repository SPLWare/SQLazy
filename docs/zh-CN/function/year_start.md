#### function: year_start
Syntax: <object_parameter> year_start
Return: first day of the year containing the date <object_parameter>.
Parameter **<object_parameter>**: Required parameter; type is date or datetime; parameter name omitted.
> Example: First day of the year containing 1980-02-27.
NLC snippet: 1980-02-27 year_start // result is date 1980-01-01
