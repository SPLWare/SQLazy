#### function: left
Syntax: left(<source_string> <position>)
Return: string. Returns the substring from the 1st to the <position>-th character on the left of <source_string>.
Parameter **<source_string>**: Required parameter; string type; parameter name omitted.
Parameter **<position>**: Required parameter; integer type; parameter name omitted.
> Example: Substring from 1st to 4th character of "中国人民电视台".
NLC snippet: left("中国人民电视台",4) // result is "中国人民"
