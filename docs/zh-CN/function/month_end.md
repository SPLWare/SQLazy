#### function: month_end
Syntax: <object_parameter> month_end
Return: last day of the month containing the date <object_parameter>.
Parameter **<object_parameter>**: Required parameter; type is date or datetime; parameter name omitted.
> Example: Last day of the month containing 1980-02-27.
NLC snippet: 1980-02-27 month_end // result is date 1980-02-29
