#### function: max
Syntax: max({<data>})
Return value: numeric. Returns the maximum among multiple numeric values.
Parameter: **<data>** original value/original expression. Usually multiple parameters used together, i.e., number of parameters >= 1. Required parameter; type is numeric; parameter name omitted.
> Example: Find the maximum of 250, 3100, 1000.
NLC snippet: max(250,3100,1000) // returns 3100
