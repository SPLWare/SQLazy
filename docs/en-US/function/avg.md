#### function: avg
Syntax: avg({<data>})
Return value: numeric. Returns the average of multiple numeric values.
Parameter: **<data>** original value/original expression. Usually multiple parameters used together, i.e., number of parameters >= 1. Required parameter; type is numeric; parameter name omitted.
> Example: Find the average of 250, 3100, 1000.
NLC snippet: avg(250,3100,1000) // returns 1450
