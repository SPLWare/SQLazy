#### function: right
Syntax: right(<source_string> <position>)
Return: string. Returns the substring from the 1st to the <position>-th character on the right of <source_string>.
Parameter **<source_string>**: Required parameter; string type; parameter name omitted.
Parameter **<position>**: Required parameter; integer type; parameter name omitted.
> Example: Substring from the 1st to 3rd character on the right of "中国人民电视台".
NLC snippet: right("中国人民电视台",3) // result is "电视台"
