#### function: first
Syntax: first({<data>})
Return value: numeric. Returns the first among multiple data/expressions.
Parameter: **<data>** original value/original expression. Usually multiple parameters used together, i.e., number of parameters >= 1. Required parameter; type is simple data type; parameter name omitted.
> Example: Take the first of "Class 2", "Class 3", "Class 3".
NLC snippet: first("Class 2","Class 3","Class 3") // returns "Class 2"
