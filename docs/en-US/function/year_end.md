#### function: year_end
Syntax: <object_parameter> year_end
Return: last day of the year containing the date <object_parameter>.
Parameter **<object_parameter>**: Required parameter; type is date or datetime; parameter name omitted.
> Example: Last day of the year containing 1980-02-27.
NLC snippet: 1980-02-27 year_end // result is date 1980-12-31
